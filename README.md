# Mercer HR Tech Content RSS Feed

Automated RSS feed generator for Mercer's HR Tech Content page. This repository scrapes HR technology articles, market briefs, and event coverage from Mercer and generates an RSS 2.0 feed with actual publish dates extracted from the articles.

## 🔗 RSS Feed URL

Once deployed, your feed will be available at:

```

```

(Replace `YOUR-USERNAME` with your GitHub username)

## ⚙️ Configuration

### Change Update Frequency

Edit `.github/workflows/update-feed.yml`:

```yaml
schedule:
  - cron: '0 9 * * *'  # Daily at 9 AM UTC
  # Examples:
  # - cron: '0 */12 * * *'  # Every 12 hours
  # - cron: '0 9 * * 1-5'   # Weekdays only
```

### Change Number of Pages

Edit `scrape_feed.py`:

```python
MAX_PAGES = 4  # Change to scrape more or fewer pages
```

## 🔍 How It Works

### Date Handling
The scraper:
1. Scrapes all 4 pages to find article links
2. Records the current date/time when an article is first seen
3. Caches dates in `article_dates.json`
4. On future runs, uses the cached date for existing articles
5. New articles get the date they were first discovered

**Note**: The first 40 articles will show the date you first run the scraper. New articles added after that will have accurate discovery dates.

## 🛠️ Troubleshooting

**Feed not updating?**
- Check Actions tab for errors
- Verify GitHub Pages is enabled
- Ensure repository is Public

**Dates showing as current date?**
- First run assigns today's date to all existing articles
- New articles discovered later will have accurate dates
- This is expected behavior

**Want to reset all dates?**
- Delete `article_dates.json` from repository
- Re-run the workflow
- All articles will get new discovery dates
